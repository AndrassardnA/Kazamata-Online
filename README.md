# HU - Kazamata Online 🎲

A **Kazamata Online** egy **nyílt forráskódú** projekt ami a **kalandjáték-könyvek** világát viszi át **webes** felületre. A projekt célja, hogy lehetőséget adjon mind **kalandok írására**, mind **végigjátszására**.

## Legfrissebb Verzió v0.1.1
**Állapot:** ADD README és CHANGELOG

## Célkitűzés
A weboldal lehetőséget nyújt más felhasználók által létrehozott kalandok végigjátszására és értékelésére. Biztosítja a rendszert amivel egy kaland végigjátszható, és a rendszert amivel a kalandok létrehozhatók. Lehetőséget ad írói profil létrehozására és más profilok értékelésére.

### Funkciók
- **Böngészés a létező kalandok között**
    - Kalandok listázása
    - Népszerűség szerinti rangsorolása
    - Típus szerinti szűrés
    - Nyelv szerinti szűrés (Magyar, Angol)
    - Név alapján való keresés
    - Kalandok értékelése
- **Kaland játékrendszer**
    - Dinamikus karakterlap
    - Harcrendszer
    - Döntések szerinti mozgás a történetben
- **Kaland létrehozása**
    - Regisztrált felhasználóknak elérhető
    - Játék rendszerének megalkotása
        - Választás a létező rendszerek közül
        - Új rendszer létrehozása
    - Történetírás
    - Történetszálak összekötése
    - Megosztás
    - Módosítás
- **Autorizáció és Profil**
    - Profil szerkesztése
    - Más profilok (mint írók) értékelése 
    - Értékelések nyilvánosan elérhetőek az oldalon

# EN - Dungeon Online 🎲

**Dungeon Online** is an **open-source** project that brings the world of **gamebooks** to a **web-based** platform. The project aims to provide an opportunity for both **writing** and **playing** adventures.

## Latest Version v0.1.1

**Status:** ADD README and CHANGELOG

## Objective

The website allows users to play and rate adventures created by others. It provides the system to play through an adventure and the framework to create them. It also allows for the creation of author profiles and the rating of other profiles.

### Features

- **Browsing existing adventures**
    - Listing adventures
    - Ranking by popularity
    - Filtering by type
    - Filtering by language (Hungarian, English)
    - Searching by name
    - Rating adventures
- **Adventure game system**
    - Dynamic character sheet
    - Combat system
    - Story progression based on choices
- **Creating an adventure**
    - Available to registered users
    - Designing the game system
        - Choosing from existing systems
        - Creating a new system
    - Story writing
    - Connecting storylines
    - Sharing
    - Modifying
- **Authorization and Profile**
    - Editing profile
    - Rating other profiles (as authors)
    - Ratings are publicly available on the site
